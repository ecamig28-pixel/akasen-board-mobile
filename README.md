# 赤入れボード Mobile

Androidでスクリーンショットへ赤入れし、完成画像を共有するためのタッチ操作版です。PC版とは別フォルダで動きます。

## 操作

1. `画像` または `画像を選ぶ` からスクリーンショットを読み込みます。
2. 下の `番号 / 丸 / 四角 / 線 / 文章` を選んで、画像をタップまたはドラッグします。
3. `選択` 中は縦スクロールできます。書き込みを一度タップすると選択され、もう一度ドラッグすると移動できます。
4. 文章は選択後、右下の青い取っ手をドラッグすると幅を変えられます。
5. 上の `共有` からAndroidの共有画面へPNG画像を渡します。共有非対応の場合はコピーまたは保存に切り替わります。

## PCでの確認

`start.cmd` をダブルクリックすると `http://127.0.0.1:8766/` で開きます。終了は `stop.cmd` です。

## Androidで使うには

このフォルダをHTTPS対応のWebサービスへアップすると、AndroidのChromeから開いてホーム画面へ追加できます。一度読み込めば、アプリ本体はオフラインでも起動できます。画像は端末内で処理され、サーバーへ送信しません。
